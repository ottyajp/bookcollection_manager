## 今できること

## udevルールの作成

- 以下の感じで、必要に応じてudevルールを作成しましょう

```
KERNEL=="ttyACM*", ATTRS{idVendor}=="065a", ATTRS{idProduct}=="a002", MODE="0666"
```
